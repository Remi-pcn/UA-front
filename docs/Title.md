`Title` (component)
===================

Displays a title

Props
-----

### `align`

Set the text-align on the component

type: `enum('inherit'|'center'|'justify'|'left'|'right')`
defaultValue: `'inherit'`


### `children` (required)

The content of the component

type: `node`


### `gutterBottom`

Enable bottom margin

type: `bool`
defaultValue: `true`


### `level`

Set content importance

type: `enum(1|2|3|4)`
defaultValue: `1`
