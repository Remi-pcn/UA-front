`button` (component)
====================

Displays a button that triggers an action when clicked

Props
-----

### `children` (required)

The button content

type: `node`


### `className`

The class of the button container

type: `string`
defaultValue: `''`


### `onClick` (required)

Function called when the user clicks on the button

type: `func`


### `primary`

Whether the button is a primary button or not

type: `bool`
defaultValue: `false`
